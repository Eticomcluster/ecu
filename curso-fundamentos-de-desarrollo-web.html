Curso Fundamentos de Desarrollo Web


Subvencionado por Red.es     /    300 horas  - 6 h día   / Presencial


Inicia o mejora tu camino como desarrollador web con este curso gratuito. Aprende a usar HTML5, CSS3 y Javascript para poder combinar estos tres lenguajes de programación en la construcción de páginas web funcionales y descubre cómo trabajar con ASP.NET y Windows Azure para desarrollar aplicaciones web.

A través de los distintos módulos, aprenderás las habilidades de programación HTML5/CSS3/JavaScript básicas para implementar la lógica de programación, definir y utilizar variables, realizar bucles y ramificación, desarrollar interfaces de usuario, capturar y validar la entrada del usuario, almacenar datos y crear una aplicación bien estructurada. Por último aprenderás a desarrollar aplicaciones ASP.NET y MVC tecnologías avanzadas y herramientas de .NET Framework 4.5, de codificar las actividades que mejoran el rendimiento y la escalabilidad de la aplicación de una página Web e implementar y de comparar formularios ASP.NET MVC frente a formularios Web.

Finalizado el curso serás capaz de diseñar y desarrollar servicios de acceso a datos remotos y locales de diversas fuentes de datos y desarrollar y desplegar servicios en entornos híbridos, incluyendo los servidores local y Windows Azure.

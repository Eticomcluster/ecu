Curso Superior en Programación de Aplicaciones con Tecnologías Full Stack en Cloud


Subvencionado por Red.es     /    300 horas  - 6 h día   / Presencial


Con este curso superior aprenderás en profundidad las tecnologías de desarrollo de aplicaciones en la nube y las arquitecturas de implementación y producción en sistemas Cloud Computing.

El proceso de aprendizaje incluye todas las tecnologías (de ahí la denominación Full Stack) empleadas en las arquitecturas de las aplicaciones: desde el lado cliente (Front-End), el lado servidor (Back-End) y las bases de datos, siempre desde el punto de vista de su implementación en la nube.

    - Para realizar este curso gratuito necesitas estar inscrito en el Sistema de Garantía Juvenil. Para más información sobre el Sistema de Garantía Juvenil, y saber cómo puedes inscribirte en el mismo, visita este enlace.

Qué incluye

Además de las horas de formación presencial, este curso también ofrece:
    - Actividades de orientación para todos los alumnos.


    - Prácticas en empresa una vez finalices.


    - Posibilidad real de contratación tras el periodo de prácticas (para al menos el 40% de los alumnos).

Programa

Módulo 1. Fundamentos de HTML5. 20 horas.
Introducción al lenguaje de marcas empleado en la web desde el punto de vista del desarrollo de las aplicaciones.
Módulo 2. Fundamentos de CSS y Frameworks CSS. 20 horas
Introducción a las hojas de estilo y frameworks como Bootstrap o Materialize, desde el punto de vista del desarrollo de aplicaciones.
Módulo 3. Fundamentos de Programación en Javascript. 20 horas
Fundamentos y código empleado en la programación en lenguaje Javascript como introducción a los frameworks basados en este.
Módulo 4. Programación de Aplicaciones con Angular JS. 50 horas
Programación con Angular JS, el framework de Javascript más empleado para el desarrollo de aplicaciones avanzadas.
Módulo 5. Programación de Aplicaciones con React JS. 20 horas
Programación con React JS, framework de Javascript orientado al Front-End
Módulo 6. Programación de Aplicaciones con Polymer. 20 horas.
Programación con Polymer, framework orientado al desarrollo de componentes de aplicaciones reutilizables.
Módulo 7. Programación de Aplicaciones móviles con Ionic. 20 horas.
Programación de aplicaciones con Ionic 2, que permite reutilizar el código Angular para las versiones móviles de aplicaciones en las plataformas iOS y Android.
Módulo 8. Bases de Datos MongoDB orientada a Big Data. 50 horas.
Programación avanzada de base de datos MongoDB no relacionales enfocada a Big Data incluyendo procesos de generación de cluster de servidores de almacenamiento masivo de datos.
Módulo 9. Programación de Servidores Cloud Node JS+Express. 10 horas.
Programación del Back-End de aplicaciones con servidores en la nube con la tecnología Node JS y Express.
Módulo 10. Frameworks Javascript orientados a la nube, Ember JS, Marionette JS, Backbone JS, Karma JS, Jasmine, Webpack y Coffescript. 20 horas.
Aprendizaje de frameworks javascript alternativos orientados a la nube.
Módulo 11. Cloud Computing en Amazon Web Services. 40 horas.
Técnicas avanzadas de cloud computing en la plataforma Amazon Web Services, desarrollo de producción de las aplicaciones en arquitecturas en la nube.
Módulo 12. Cloud Computing en Digital Ocean. 10 horas.
Técnicas avanzadas de cloud computing en la plataforma Amazon Web Services, desarrollo de producción de las aplicaciones en arquitecturas en la nube.

Curso Superior en Programación de Aplicaciones con Tecnologías Java en Cloud


Subvencionado por Red.es     /    300 horas  - 6 h día   / Presencial


Con este curso aprenderás en profundidad el desarrollo de proyectos de aplicaciones Java para su implementación como arquitecturas en sistemas Cloud Computing.

El proceso de aprendizaje incluye el conocimiento avanzado del lenguaje Java, el desarrollo de componentes para aplicaciones, las herramientas y frameworks de desarrollo más importante y la implementación en la nube.

Los contenidos incluyen el curso Java SE 8 Fundamentals de Oracle (módulos 1 a 12) y el curso Java SE 8 Programming (módulos 12 a 25), ambos conducentes a certificaciones oficiales de Oracle.

Qué incluye
